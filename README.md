## How to Run and Set Up This Project

1. **Download the Bootstrap Utility:** Download the automated [Setup Script](https://github.com/lhldev/late-to-class/releases/download/setup-script/SetupProject.bat) to your local machine.
2. **Execute the Configuration:** Run the downloaded `.bat` by double-clicking the file. For more details about the script, you can review the [Setup Script Release Notes](https://github.com/lhldev/late-to-class/releases/tag/setup-script).
3. **Initialize the Workspace:** Once the script is successfully ran, open the newly generated folder that is located in the same folder as the script called "late-to-class" within Visual Studio Code application.
4. **Compile and Execute:** Press **Ctrl + Shift + B** to build and run the C++ project.

The game code is in the src folder
